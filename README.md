# Financial Accounting

> 純前端個人財務管理工具，所有資料儲存於瀏覽器 IndexedDB，**完全離線可用，無需後端伺服器**。

---

## 五大功能

### 1. 投資紀錄表
管理股票、基金等投資資產與帳戶，完整追蹤買賣損益。

- 資產管理：新增/編輯/刪除投資標的（台股、美股、日股、基金），支援一鍵即時更新股價
- 帳戶管理：管理券商/銀行/現金帳戶，支援跨幣別帳戶轉帳
- 交易紀錄：支援買入、賣出、股息、手續費、稅費、入金、出金 7 種類型，採 FIFO 計算賣出損益
- 持倉統計：自動計算總投入、總賣出、已實現損益、未實現損益（台股慣例紅漲綠跌）
- 即時匯率：從 `open.er-api.com` 更新 USD/JPY/CNY 對 TWD 匯率

### 2. 資產再平衡表
依資產類型或幣別，視覺化呈現目前配置比例並提供再平衡建議。

- 圓餅圖顯示目前各類別佔比（成本配置 vs 市值配置）
- 設定各類別目標比例與容許誤差，自動標示是否需要調整
- 計算達到目標比例所需買入/賣出的 TWD 金額

### 3. 收入與日常消費表
追蹤日常收支、設定每月預算計畫，分析月度收支趨勢。

- 月度總覽：收支趨勢長條圖、支出分類圓餅圖（依金額排序）、月度摘要表
- 收支紀錄：含幣別、分類、現金帳戶連結，儲存後自動更新帳戶餘額；切換幣別時自動帶入參考匯率
- 固定月計畫：分別設定固定收入/支出計畫，一鍵批次匯入為當月實際紀錄
- 分類管理：自訂收入/支出分類，內建 16 個預設分類
- 現金帳戶 (TWD) 卡片可點擊，展開各帳戶餘額構成明細

### 4. 退休規劃試算
以蒙地卡羅模擬預測退休資產變化與成功機率。

- 輸入：目前年齡、退休年齡、預期壽命、目前總資產、每月投入金額、預期年化報酬率與波動率、退休後月支出、通膨率、模擬次數
- 帶入按鈕：可一鍵帶入再平衡表總市值（目前總資產）與最新月結餘（每月投入）
- 輸出：退休成功率、退休時資產中位數、資產耗盡年齡中位數、百分位資產走勢圖（P10/P25/P50/P75/P90）

### 5. 投資績效分析
自動彙整投資資產的報酬率與配息數據，提供多維度績效診斷。

- 一般標的分析表：成本、市值、未實現損益、累積股息（從「股利」收支紀錄自動加總，原幣計算）、含息總報酬率
- 基金分析表：淨值損益、累積淨配息、配息率%、含息總報酬、近12月配息、近12月殖利率%、成本殖利率%（均以原幣分子/原幣分母計算，避免匯率失真）
- 摘要 KPI 卡片：總市值、總成本、未實現損益、累積配息（TWD）
- 風險警示分頁：自動偵測淨值跌幅 >15%/25%、單一標的占比過高等問題
- 篩選面板：依幣別、是否只顯示持有中等條件篩選；表格欄位可排序
- 一鍵更新現價（Yahoo Finance / TWSE / Stooq 多來源）

### 6. 資料工具
提供資料品質健康檢查與 JSON 備份管理。

- 備份強化：標籤備份、下載 JSON、備份歷史記錄（最多 20 筆）；≥30 天未備份時顯示警示
- 資料健康檢查：自動偵測 9 種資料問題（孤立交易、重複 Ticker、帳戶餘額異常等）

## 離線使用方式

### ⭐ 方式一：一鍵本機啟動（推薦，無需安裝任何套件）

> 前提：已安裝 [Node.js](https://nodejs.org)（僅需 Node.js 本身，不需 npm install）

下載並解壓縮 [`dist.zip`](dist.zip) 後，直接雙擊：

```bash
start-offline.bat
# → 自動開啟瀏覽器 http://localhost:8080
```

也可以在專案目錄執行：

```bash
node serve.cjs
# → 自動開啟瀏覽器 http://localhost:8080
```

此模式會啟動本機 API Proxy，更新股價 / 匯率最穩定。按 `Ctrl+C` 停止。

### 方式二：npm preview（需先 npm install）

```bash
npm run build        # 打包 → dist/
npm run preview      # 預覽 → http://localhost:4173
```

### 方式三：部署 `dist/` 目錄至任意靜態伺服器
將 `dist/` 目錄內的所有檔案部署至 Apache、Nginx、GitHub Pages 等即可。

> 已提供完整離線包 [`dist.zip`](dist.zip)，內含 `dist/`、`serve.cjs`、`start-offline.bat`。解壓縮後建議雙擊 `start-offline.bat` 使用；若只雙擊 `dist/index.html`，基本介面可用，但更新股價會受瀏覽器 `file://` CORS 限制。

---

## 打包輸出檔案（`dist/`）

| 檔案 | 說明 |
|------|------|
| `dist/index.html` | 入口 HTML（約 0.4 KB） |
| `dist/assets/index-*.js` | 所有 JS + CSS 程式碼（約 990 KB，gzip 後約 270 KB） |

> IIFE 格式打包，CSS 內嵌於 JS 中，支援 `file://` 直接開啟與 `node serve.cjs` 本機伺服器兩種模式。
>
> 所有資料僅存於使用者瀏覽器的 IndexedDB，清除瀏覽器資料前資料不會遺失。
> 可使用內建「匯出 JSON」功能定期備份資料。

---

## 使用限制

| 功能 | `start-offline.bat` / `node serve.cjs` | 雙擊 `index.html` |
|------|:---:|:---:|
| 基本介面 | ✅ | ✅ |
| 更新股價 / 匯率 | ✅（內建本機 Proxy） | ❌（受瀏覽器 `file://` CORS 限制） |
| 資料儲存（IndexedDB） | ✅ | ✅ |

> 建議使用 `start-offline.bat` 或 `node serve.cjs` 以獲得完整功能；不要直接從 `file://` 更新股價。

## 本機開發

```bash
npm install          # 安裝依賴
npm run dev          # 開發伺服器（含股票 API Proxy）
npm run build        # 正式打包
```
